## goals

This course will focus on object-oriented programming, software design and engineering principles, formal specification, testing, and design patterns

- Write programs in Java under the OO paradigm
- Choose and justify certain designs over others
- Understand software engineering principles
- Create a module from a formal specification
- Write a formal specification for a particular module
- Understand different methods of testing and their importance
- Identify appropriate design patterns which apply to a specific problem
