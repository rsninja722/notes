# studying techniques

## low/zero notes method

[reference](https://www.youtube.com/watch?v=JNJJEFzaOiA)

- read material first and create flashcards
- watch lectures on 2x to catch any missed ideas
- employ [SQRRR](https://en.wikipedia.org/wiki/SQ3R)
    - **survey** - skim through a chapter and identify key concepts 
    - **question** - generate questions about content
    - **read** - read in order to answer the questions created
    - **recite** - recall information learned
    - **review** - review passages completed
