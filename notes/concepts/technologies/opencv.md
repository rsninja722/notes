## installation

1. Choose a build from [sourceforge](http://sourceforge.net/projects/opencvlibrary/files/) you want to use and download it.
2. Make sure you have admin rights. Unpack the self-extracting archive.
3. You can check the installation at the chosen path as you can see below.
4. set your environment variables ( `setx OpenCV_DIR C:\Code\c++\opencv\build\x64\vc16` )
