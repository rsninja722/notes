# types and operator notation

- **operator notation** - $\_ = \_ :  t_1 \times t_2 \rightarrow t_3$ - means an operator (equals) is a binary infix operator that takes type $t_1$ and type $t_2$ as the arguemtents, and results in type $t_3$


## types

- **boolean value type** - $\mathbb{B}$
    - used to specify the type of propositions
