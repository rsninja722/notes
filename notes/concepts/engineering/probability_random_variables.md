# random variables

- **random variable** - a function from $\Omega$ into the real numbers
    - the value of a random variable $X$ at sample point $w$ is $X(w)$
    - **degenerate** - there exists a real value *b* such that $P(X=b)=1$
    - **discrete** - there exists a set $\\{k_1,k_2,...\\}$ such that $\sum_i P(X = k_i) = 1$
- **probability distribution** (of the random variable $X$) - the collection of probabilities $P \\{X \in B\\}$ for sets $B$ of real numbers
- **PMF probability mass function** - $p(k) = P(X = k)$ for the possible values $k$ of $X$
    - example: for rolling two die, the probability that the sum is 5: $p(5) = \frac{4}{36}$
