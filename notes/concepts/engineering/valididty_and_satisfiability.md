# valididty and satisfiability

- **satisfied** - a boolean expression that evaluates to *true* in state *s*
- **satisfiable** - a boolean expression for which there exists a state that makes it satisfied
- **valid** - a boolean expression that is satisfied in every state
- **tautology** - valid boolean expression
- **contradiction** - a boolean expression that evaluates to *false* in every state
- **logical equivalent** - two boolean expressions that evaluate to the same value in every state
