# boolean logic and logic gates

- **boolean constants** - true,false
	- also shown as T,F or 1,0
- **truth table** - a table of outputs for all combinations of inputs
	- the column headers can correspond to variables in a proposition, or labels on a circuit diagram
- **logic gate** - an electrical component that takes in some input values, and according to a logical proposition, outputs some values
