# programming paradigms

- **object oriented** - everything is an object
    - c#, c++, python, java
- **functional** - no side effects
    - scala, haskell, elm, ocaml, f#
- **imperative/procedural** - step by step machine instructions
    - c, and the rest
- **logical** - prolog

- models of computation
    - turing machines
    - lambda calculus
